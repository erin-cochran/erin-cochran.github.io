about

My name is Erin Cochran. I'm a front-end web developer that's been building webpages for fun since the 1990s, and professionally for around 10 years. I currently reside in Portland, Oregon. Before I lived here I lived in Austin, Texas.

I started using linux when I was 14 because I got sold a computer with a pirated copy of windows on it and doing a windows update caused a bunch of problems, so I went to a software store (we had those back then) and purchased a bundle of Red Hat Linux disks. The rest is history.

I enjoy running, riding bikes, self hosting things, collecting vinyl, and tinkering with old computers.

My current workstation is a ThinkPad P14s running Arch Linux. By the way. My phone is a Pixel 8 running GrapheneOS because I'm paranoid.
